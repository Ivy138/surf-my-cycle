## 待办计划（Roadmap）

### 1. 优化AI对话 System Prompt
- **优先级**：高
- **内容**：
  - 让AI更了解用户月经周期阶段
  - 根据不同时段（早上/下午/晚上）给出不同回应
  - 优化情绪评分格式，减少解析错误
  - 让AI记住用户的历史偏好

### 2. 打开自动跳转当天
- **优先级**：高
- **状态**：✅ 已完成
- **解决**：使用 `toLocaleDateString('en-CA')` 修复UTC时间导致的日期偏差

### 3. AI记忆架构（参考OpenClaw）
- **优先级**：高
- **内容**：
  - 多层记忆模式：
    - 短期记忆：当日AI对话（smc_ai_YYYY-MM-DD）
    - 长期记忆：用户偏好、重要原则（smc_memory）
    - 语义搜索：基于关键词检索历史对话
  - AI自动优化系统提示词（在无风险情况下）
  - 对话结束后自动生成摘要存入记录

### 3. 页面设计优化
- **优先级**：中
- **内容**：
  - AI对话界面美化（气泡样式、动画）
  - 评分标签视觉优化
  - 移动端适配检查
  - 整体配色一致性

### 3. 数据存储可靠性
- **优先级**：高
- **内容**：
  - localStorage容量检查
  - 数据导出/备份功能
  - 数据格式版本兼容
  - 异常情况处理

### 4. 账号系统与云端同步（待完善）
- **优先级**：高
- **状态**：⏸️ 暂停（需重新设计）
- **问题**：
  - 本地账号密码登录与游客模式冲突
  - 用户期望"打开即用"，但需要云端同步
  - 需要重新梳理用户流程：游客模式 → 可选注册/登录 → 数据迁移
- **方向**：考虑直接使用 Supabase Auth 简化流程

### 5. 后续功能（未来）
- [ ] AI历史记录长期记忆
- [ ] 周期预测增强
- [ ] 数据图表丰富
- [ ] 导出PDF/分享功能
