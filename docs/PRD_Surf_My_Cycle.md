# 🌊 Surf My Cycle - 产品需求文档

> 一个帮助你了解自己周期规律的个人工具  
> 通过每天记录,发现什么时候效率高、什么时候容易崩溃  
> **基于科学研究,但尊重个体差异**

---

## 🔬 科学基础

这个产品不是随意设计的,而是基于真实的科学研究:

**核心科学发现**
- 周期确实影响情绪、能量和认知功能 (Sundström Poromaa & Gingnell, 2014)
- 黄体中期(高孕激素期)在某些认知任务上有优势 (Xu et al., 2022)
- 情绪变化比认知变化更明显和一致
- **但个体差异非常大** - 每个人的模式可能完全不同

**我们的方法**
- ✅ 追踪科学证明有影响的指标(情绪、能量、效率)
- ✅ 长期记录发现个人规律(至少3个周期)
- ✅ 不套用"标准周期"模板
- ✅ 用你自己的数据说话

> 详细科学依据请查看: `docs/科学基础.md`

## 📖 产品定位

**为谁做**: 为你个人量身定制  
**解决什么**: 监控周期、了解激素变化、预测状态、优化生活安排  
**核心价值**: 认识自己,与自己的周期和谐共处

---

## 🎯 要实现什么

1. **记录**: 每天快速记录生理、情绪、能量、效率等状态
2. **可视化**: 用日历和图表看清整个周期的变化规律
3. **分析**: 自动识别你的周期模式和高低谷期
4. **洞察**: 告诉你什么时候效率高、什么时候需要休息
5. **预警**: 提前预测可能的低谷期,帮你做好准备

## 📱 核心功能

### 1. 每日记录
- 生理状态(月经期、排卵期等)
- 情绪评分(1-10)
- 能量水平(1-10)
- 工作效率(1-10)
- 社交欲望(1-10)
- 食欲/渴望
- 自由备注

**要求**: 3分钟内完成,支持补记

### 2. 周期日历
- 日历视图展示每天的状态
- 用颜色区分不同阶段
- 点击查看详细记录
- 可对比不同月份

### 3. 数据分析
- 周期长度统计
- 情绪/能量/效率曲线
- 相关性分析
- 用图表和文字呈现发现

### 4. 个人洞察
- 你的高效期是哪几天
- 低谷期通常在什么时候
- 激素变化对你的影响
- 个性化的行动建议

### 5. 预警系统
- 提前3-7天预测状态
- 标注可能的崩溃风险
- 给出应对建议

### 5. 周期感知校准系统 ⭐ 核心功能
**解决核心问题**：用主观感受定义自己的周期，而非套用标准生理周期

#### 体感标记（核心）
每天记录主观感受：
- **能量水平**（1-10）：今天整体精力如何
- **专注力**（1-10）：能否长时间专注工作
- **社交欲望**（1-10）：想见人还是想独处
- **食欲/渴望**（1-10）：对食物或特定食物的渴望程度
- **性欲**（1-10）：身体亲密欲望
- **情绪基调**（1-10）：整体情绪状态

#### 简单修正机制
- 每天记录时，若体感与系统预测不符，直接点击自选阶段覆盖
- 系统记录修正历史，用于优化个人模型
- 无需额外填写理由或评分，保持记录流程简洁

#### 个人周期模式学习
- **不套用28天模板**：根据你的主观数据建立个人能量周期
- **识别你的规律**：
  - "你通常在每月第X天进入高效期"
  - "你的低谷期平均持续X天"
  - "你的性欲高峰与能量高峰的关系"
- **预测置信度**：显示"基于你的历史数据，这个预测我有X%把握"

#### 行动建议引擎
根据你的周期模式给出建议：
- **高效期**：适合深度工作、重要会议、创造性任务
- **过渡期**：适合整理、计划、轻度社交
- **低谷期**：适合休息、独处、轻度事务、自我关怀

#### 修正流程
```
系统预测高能量期 → 你标记"感觉很疲惫" → 记录真实状态 → 
算法学习调整 → 下次更准确预测你的个人周期 → 显示"基于你的反馈"
```

## 🎨 界面设计

**原则**: 极简、快速、清晰

### 主页布局（日历为核心）

**设计理念**：参考美柚APP，采用**左右固定分栏**结构

**左侧 - 月历视图（60%）**
- 完整月历，显示当月所有日期
- 每个日期格子显示：
  - 日期数字
  - 周期阶段标识（用颜色区分：🔴月经期 🟠卵泡期 🩷排卵期 🟣黄体期）
  - 记录状态指示点（已记录/未记录）
- 支持月份前后切换
- 今日日期特殊高亮

**右侧 - 记录面板（40%）**
- **顶部**：选中日期信息
  - 日期显示（如：3月21日 周四）
  - 周期Day显示（如：Day 15）
  - **系统预测阶段**：显示算法预测的当前阶段（如：排卵期）
  - **用户自选阶段** ⭐ 核心交互：
    - 一排阶段按钮：🔴月经期 🟠卵泡期 🩷排卵期 🟣黄体期 ？不确定
    - 用户可点击选择"我觉得我今天处于哪个阶段"
    - 若选择与系统预测不同，标记为"体感修正"，用于算法学习
    - 可选：填写简短理由（如"今天突然很疲惫，感觉已经过了"）
- **中部**：快速记录区
  - 💬 聊天式输入框（自然语言记录）
  - ✨ AI自动填充建议
  - 5个核心评分滑块（心情/能量/效率/社交/食欲）
- **底部**：详细选项（可折叠）
  - 生理状态选择
  - 食物渴望勾选
  - 行为特征勾选
  - 备注输入
- 💾 保存按钮

**交互流程**
```
进入主页 → 看到本月日历 + 今日记录面板（默认选中今天）
点击日历中的某一天 → 右侧面板切换到该日期
已有记录 → 显示历史数据（可编辑）
无记录 → 显示空白表单（可填写）
填写/修改 → 保存 → 日历更新指示点
```

**响应式适配**
- PC端（>900px）：左右分栏 6:4
- 平板端（600-900px）：左右分栏 5:5
- 移动端（<600px）：上下堆叠，日历在上

**主要页面**:
1. 日历主页（记录 + 查看）
2. 数据分析
3. 个人洞察

## 📊 追踪指标

**生理**: 月经周期、体温、身体症状  
**心理**: 情绪、压力、焦虑感  
**行为**: 效率、社交、睡眠、食欲  
**激素**: 雌激素、孕激素、睾酮的感受

## 💡 使用场景

- **每天**: 记录当天状态
- **每周**: 查看情绪曲线
- **每月**: 对比周期规律
- **重要事项**: 根据预测安排时间

## 📝 产品原则

1. **简单至上** - 只做有用的功能
2. **尊重数据** - 不美化、不修改真实记录
3. **个性化** - 发现你独特的模式,不套用标准
4. **可操作** - 给出能指导生活的建议
